Place your landing-page slideshow images in this folder with these exact names:

1) chakali-slide-1.jpg
2) chakali-slide-2.jpg
3) chakali-slide-3.jpg
4) chakali-slide-4.jpg

Recommended image size (responsive):
- Width: 1600 px
- Height: 900 px
- Format: JPG

The homepage slideshow will automatically use these images first.
If these files are missing, it falls back to product upload images.
